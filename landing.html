Key Projects

Here are some of the key projects I've worked on, showcasing my skills in AI, machine learning, and computer vision.

AI and ML in Healthcare and Finance

Developed machine learning models for early Parkinson's disease diagnosis and money laundering detection. The healthcare model improved techniques for early diagnosis of Parkinson's disease through computer vision techniques to recognize reduced facial muscle movements with a 95 percent classification accuracy. The financial model improved techniques of recognizing money-laundering when observing bank transactions using feature engineering to reduce false positives as compared to traditional rules based programs.

Researched computer vision through 3 phases using Raspberry Pi computer and OpenCV programming library. The project focused on facial identification within photos and videos, object identification in videos, and object detection on live video feed through a pan-tilt camera assembled with servos. This introductory project to AI and ML focused on using deep learning-based object detection algorithms and neural networks.
